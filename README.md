## BuilderApp

### Usage

```
npm install
npm install -g webpack-dev-server
webpack-dev-server
```

### Build and Deploy

```
webpack (for test, not minified) / webpack -p (minified for production)
firebase deploy
```

### Dependencies

```
AngularJS
Angular UI Router
AngularFire
AngularAnimate
AngularMoment
Bootstrap
Firebase
Filestack
jQuery
jsPDF
MomentJS
webpack
less

```
